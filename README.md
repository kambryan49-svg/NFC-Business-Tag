# NFC Business Tag

A modern, responsive website for NFC Business Tag that provides quick access to social media profiles and contact information.

## Features

✨ **Social Media Links**
- Facebook
- Instagram
- TikTok
- Email

📱 **Contact Features**
- Save Phone Contacts (vCard format)
- WhatsApp QR Code Scanner
- Looping Animation

🎨 **Design**
- Beautiful gradient backgrounds
- Smooth animations and transitions
- Fully responsive design
- Mobile-friendly interface

## Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
3. Customize the links and contact information in the files

## Customization

### Update Social Media Links
Edit the `href` attributes in `index.html`:
- Facebook: Change `https://www.facebook.com/`
- Instagram: Change `https://www.instagram.com/`
- TikTok: Change `https://www.tiktok.com/`
- Email: Change `your-email@example.com`

### Update Contact Information
In `index.html`, modify:
- Contact Name: `Your Name`
- Phone Number: `+1 234 567 8900`
- Email Address: `your-email@example.com`

### Update WhatsApp QR Code
Replace the phone number in the WhatsApp QR code URL:
```
https://wa.me/1234567890
```
Change `1234567890` to your WhatsApp number (without + or spaces)

## Features Explained

### WhatsApp QR Code
- Displays a QR code that can be scanned to start a WhatsApp conversation
- Also includes a direct WhatsApp link

### Save Contact
- Generate a vCard (.vcf) file that can be imported into contact lists
- Downloads automatically when clicked

### Looping Icon
- Interactive sync icon that rotates when clicked
- Creates an engaging loading effect
